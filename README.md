# UniVilm

Platform Streaming Film Sederhana berbasis Laravel.

## Fitur
- Login & Register
- Home
- Trending Movies
- Search Movie
- Detail Movie
- Profile User

## Teknologi
- Laravel
- PHP
- MySQL
- Blade
- Tailwind CSS
- Vite

## Cara Menjalankan

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
